## React Redux

### Required 

* node v6.8.1 or v6.6.0
* webpack v3.0.0

### Usage

* Run `npm start`

### Something You Need to Know

* 该项目最初是以 script 标签的方式来引入所有依赖的库的，但由于浏览器不支持 import/export 等语法，所以使用了 webpack。
